//tab
:newtab filename
gt   (go prev tab)
gT    (go next tab)
(numPad) gt  (go to tab #)

//window
vsplit filename - split currently window into two vertically and open a file
<C-w>n - :new horizontal split (editing a new empty buffer)
<C-w>s - :split window horizontally (editing current buffer)
<C-w>v - :vsplit window vertically (editing current buffer)
<C-w>c - :close window
<C-w>o - close all window, leaving :only the current window open
<C-w>w - go to next window
<C-w>p - go to previous window
<C-w><Up> - go to window above
<C-w><Down> - go to window below
<C-w><Left> - go to window on left
<C-w><Right> -go to window on right

//find and replace
:%s/<search_string>/<replace_string>/g   find and replace for entire file 
g => global search
c => Ask for confirmation first
i => Case insensitivee
